Mental model in one sentence: the website sells subscriptions through Stripe, Stripe’s webhooks insert rows into substrate_provision_queue / substrate_tier_changes / destroy_queue, and workers on the compute box drain those queues and walk the substrates state machine (provisioning → running → read_only → suspended → deprovisioned). Workers are the only things that move state forward.
